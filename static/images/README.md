# Vehicle Images

Place vehicle images in this directory with the following naming convention:

- `car.jpg` - Default car image
- `motorbike.jpg` - Default motorbike image
- `truck.jpg` - Default truck image
- `default.jpg` - Fallback image for all vehicle types

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 800x600 pixels
- **Aspect Ratio**: 4:3
- **File Size**: < 500KB for optimal loading

## Using Custom Images

You can use any placeholder image service or your own images. For quick setup, you can use:

1. https://via.placeholder.com/800x600/667eea/FFFFFF?text=Car
2. https://via.placeholder.com/800x600/764ba2/FFFFFF?text=Motorbike
3. https://via.placeholder.com/800x600/28a745/FFFFFF?text=Truck
4. https://via.placeholder.com/800x600/6c757d/FFFFFF?text=Vehicle

Or download free stock images from:
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Pixabay (https://pixabay.com)

Search for:
- "car"
- "motorcycle" or "motorbike"
- "truck" or "delivery truck"
